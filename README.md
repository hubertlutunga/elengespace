# Elenge Space

Site web de la webradio Elenge Space.

## Structure

- `index.php` : point d’entrée du site.
- `pages/` : pages publiques.
- `admin/` : interface d’administration.
- `includes/` : fonctions et helpers PHP.
- `storage/content.json` : contenu éditable du site.
- `PHPMailer/` : bibliothèque d’envoi d’e-mails.

## Notes

Le fichier `storage/users.json` est ignoré volontairement car il contient des données locales/sensibles.
